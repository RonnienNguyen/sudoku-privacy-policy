# Sudoku Privacy Policy

This repository hosts the privacy policy for the Sudoku app, published via GitHub Pages.

## Setup Instructions

1. **Create the GitHub Repository**
   - Go to [github.com](https://github.com)
   - Create a new repository named: `sudoku-privacy-policy`
   - Make it public (required for GitHub Pages)

2. **Push These Files**
   - Clone the repository locally
   - Copy `index.md` to the repository
   - Commit and push to GitHub

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Navigate to **Settings** â†’ **Pages**
   - Under "Source", select **Deploy from a branch**
   - Choose **main** branch and **/ (root)** folder
   - Click **Save**

4. **Get Your URL**
   - Your privacy policy will be available at:
   - `https://yourusername.github.io/sudoku-privacy-policy/`

5. **Use in Play Console**
   - Copy the URL above
   - Paste it into **Privacy policy URL** in Google Play Console

## Customization

Edit `index.md` to customize the privacy policy content according to your app's specific data collection and usage practices.
